Task Management App
A Flutter-based Task Management App  allows users to create, edit, delete, and categorize tasks, mark them as completed, and save them persistently using SharedPreferences. The app includes multiple screens, a splash screen, and an intuitive, modern UI.

 Features:-

Splash Screen: Displays app branding for 3 seconds before navigating to the main task screen.

Task Management:
Add new tasks with title, category, and priority.
Edit existing tasks.
Delete tasks.
Mark tasks as completed (tasks show a strikethrough when done).

Task Categories:
Work, Study, Personal, Finance, Home, Health, Important.

Task Priorities:
High, Medium, Low (highlighted with color borders).

Persistent Storage:
All tasks are saved locally using SharedPreferences, so tasks remain after app restart.

Intuitive UI:
Modern design with input fields, dropdowns, buttons, and list view for tasks.

 Screens:-
Splash Screen
Displays app icon, name, and tagline.
Task Management Screen
Main screen to view, add, edit, delete, and mark tasks completed.
Category & Priority Filters
Choose task category and priority when adding a task.

Technologies Used:-
Flutter – Cross-platform UI development.
Dart – Programming language for Flutter.

SharedPreferences – Persistent storage for saving tasks locally.
Material Design – For modern app UI elements.
